Requisitos del sistema

Para ver este sitio web correctamente, asegúrese de que su sistema cumpla con los siguientes requisitos:

Requisitos del navegador

  • Google Chrome (versión 90 o posterior)
  • Mozilla Firefox (versión 88 o posterior)
  • Microsoft Edge (versión 90 o posterior)
  • Safari (versión 14 o posterior)
  • Cualquier navegador moderno con soporte HTML5 y CSS3

Requisitos del sistema

  • Conexión a internet
  • JavaScript habilitado
  • Resolución de pantalla: 1024x768 o superior

Si encuentra algún problema, intente actualizar su navegador o habilitar JavaScript.
